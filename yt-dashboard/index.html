One-time console setup (same project as before):
1. Enable YouTube Data API v3 and YouTube Analytics API (API Library).
2. APIs & Services → OAuth consent screen → External → fill app name/email → add yourself as a Test user.
3. Credentials → Create Credentials → OAuth client ID → type Web application → under Authorized JavaScript origins add the exact URL this page is served from (e.g. https://yourname.github.io or http://localhost:8000).
4. Copy the Client ID into the box above. Tip: append ?client_id=YOUR_ID to this page's URL to prefill it.
This page must be opened via http(s) — Google sign-in doesn't work from a file:// path.
